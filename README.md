# letter
A letter guessing game (a-z)

## Features

- Simple
- Fun to play
- Completely random letter

## Requirements

- Python 3.x

## Installation

Run in Terminal using **chmod +x /path/to/Letter Guessing Game.py**
Then, use **./path/to/Letter Guessing Game**
